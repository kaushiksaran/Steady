To run this extension locally for judging and evaluation:



Download (zip file)/Clone this repository to your local machine.



Extract zipped folder.



Open Google Chrome and navigate to chrome://extensions/.



Enable "Developer mode" using the toggle in the top-right corner.



Click the "Load unpacked" button in the top-left.



Select the root folder of this project (the one containing manifest.json).



The Steady icon will appear in your toolbar. Ensure it is enabled.



Important: Refresh any open tabs to allow the content scripts to inject.
